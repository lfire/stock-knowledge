# 常用技术指标

技术指标是对价格、成交量数据进行数学运算后得到的辅助分析工具。以下是最常用的几类。

## 移动平均线（MA）

将一段时间的收盘价取平均，平滑价格波动，识别趋势方向。

- **常用周期**：5 日、10 日、20 日、60 日、120 日、250 日。
- **金叉**：短期均线上穿长期均线，视为买入信号。
- **死叉**：短期均线下穿长期均线，视为卖出信号。

## MACD（指数平滑异同移动平均线）

由快线（DIF）、慢线（DEA）和柱状图（MACD 柱）组成，用于判断趋势与动能变化。

- DIF 上穿 DEA → 金叉，偏多。
- DIF 下穿 DEA → 死叉，偏空。
- 柱状图由负转正 / 由正转负，反映动能切换。

## KDJ（随机指标）

由 K、D、J 三条线组成，对短期超买超卖较为敏感。

- **超买区**：K、D 在 80 以上，警惕回调。
- **超卖区**：K、D 在 20 以下，可能反弹。

## RSI（相对强弱指标）

衡量价格涨跌力量的对比，取值 0–100。

- RSI > 70：超买，回调风险增加。
- RSI < 30：超卖，反弹概率增加。

## 成交量

"量在价先"，成交量是验证价格走势的重要依据：

- **量价齐升**：上涨健康，趋势可靠。
- **价升量缩**：上涨乏力，需警惕。
- **放量滞涨**：可能见顶。

## 使用建议

::: warning 注意
任何单一指标都有滞后性和局限性，应**多指标互相印证**，并结合趋势、形态与基本面综合判断，切勿机械照搬信号。
:::
