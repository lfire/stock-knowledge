# 风险管理

> 投资中最重要的不是赚多少，而是**活得久**。控制风险是长期复利的前提。

## 为什么风险管理至关重要

亏损与回本是不对称的：

| 亏损幅度 | 需要的回本涨幅 |
| --- | --- |
| -10% | +11% |
| -30% | +43% |
| -50% | +100% |
| -90% | +900% |

可见，**控制最大回撤**比追求高收益更能保护本金。

## 仓位管理

- **不满仓、不借钱**：保留现金应对极端情况，杜绝杠杆爆仓风险。
- **分散持仓**：避免单只个股占比过高，降低非系统性风险。
- **金字塔加仓 / 减仓**：上涨逐步减仓、下跌分批建仓，避免一次性押注。

## 止损与止盈

- **止损**：设定可承受的亏损上限（如 -8%），触发即离场，防止小亏变大亏。
- **止盈**：在达到目标或逻辑兑现后兑现利润，避免利润回吐。
- 止损止盈应在买入**之前**就规划好，而非临场情绪化决定。

## 分散与配置

- **跨资产**：股票、债券、现金、商品等合理配置。
- **跨行业**：避免集中在单一行业，降低行业系统性风险。
- **定投**：通过定期定额摊薄成本，平滑市场波动。

## 心态管理

- 接受"不可能买在最低、卖在最高"。
- 避免追涨杀跌与频繁交易。
- 用闲钱投资，避免影响生活的资金压力扭曲决策。

::: tip 一句话总结
先求不败，再求胜。**保住本金，是投资的第一原则。**
:::
