# 股票知识库

一个基于 [VitePress](https://vitepress.dev/) 构建的股票知识博客站点，通过 GitHub Pages 托管展示。内容涵盖股票基础知识、技术分析、基本面分析与投资策略。

## 功能特性

- 📈 体系化的股票知识内容（基础 / 分析 / 策略）
- ✍️ **博客式布局**：文章按发布时间倒序展示，含日期、标签与摘要
- 🏷️ **标签分类**：博客页支持按标签一键筛选文章
- 🎨 **财经红金主题**：自定义品牌配色与首页渐变
- 💬 **评论功能**：基于 [giscus](https://giscus.app/zh-CN)（GitHub Discussions，免费、无需服务器）
- 📊 **访问统计**：基于 [vercount](https://vercount.one)（兼容不蒜子，免注册）展示站点 PV/UV 与单页阅读量
- 🔍 本地全文搜索、深色模式、移动端适配

## 技术栈

- [VitePress](https://vitepress.dev/) —— 静态站点生成器
- [GitHub Pages](https://pages.github.com/) —— 站点托管
- [GitHub Actions](https://docs.github.com/actions) —— 自动化构建与部署
- [giscus](https://giscus.app/zh-CN) —— 评论系统
- [vercount](https://vercount.one) —— 访问量统计

## 本地开发

本项目使用 [pnpm](https://pnpm.io/) 作为包管理器（也可使用 npm）。

```bash
# 安装依赖
pnpm install

# 启动本地开发服务器（默认 http://localhost:5173/stock-knowledge/）
pnpm run dev

# 构建生产版本
pnpm run build

# 本地预览构建产物
pnpm run preview
```

## 部署到 GitHub Pages

1. 在 GitHub 创建名为 `stock-knowledge` 的仓库，并推送本项目代码到 `main` 分支。
2. 进入仓库 **Settings → Pages**，将 **Build and deployment → Source** 设置为 **GitHub Actions**。
3. 之后每次推送到 `main` 分支，工作流会自动构建并部署。
4. 部署完成后访问：`https://<你的用户名>.github.io/stock-knowledge/`。

> **注意**：`docs/.vitepress/config.mts` 中的 `base` 已设置为 `/stock-knowledge/`，需与仓库名保持一致。若部署到 `<用户名>.github.io` 根仓库，请将 `base` 改为 `'/'`。

## 内容贡献

所有内容以 Markdown 编写，存放于 `docs/` 下对应栏目目录。新增文章后，在 `docs/.vitepress/config.mts` 的 `sidebar` 中添加对应链接即可。

### 博客文章 frontmatter 规范

在 `docs/articles/` 下新增文章时，请在文件顶部添加如下 frontmatter，博客列表页会自动按 `date` 倒序收录并展示标签与摘要：

```markdown
---
title: 文章标题
date: 2026-06-30        # 用于排序，必填
tags: [新手入门, 实战]   # 标签，用于筛选
description: 一句话摘要，显示在列表中
---
```

## 功能配置

### 评论（giscus）

评论默认处于"未配置"提示状态，按以下步骤启用：

1. 在 GitHub 仓库 **Settings → General → Features** 中开启 **Discussions**。
2. 安装 [giscus App](https://github.com/apps/giscus) 并授权当前仓库。
3. 打开 [giscus.app/zh-CN](https://giscus.app/zh-CN)，填入仓库信息，复制生成的 `data-repo`、`data-repo-id`、`data-category`、`data-category-id`。
4. 将这些值填入 `docs/.vitepress/theme/giscus.config.ts`。

### 访问统计（vercount）

访问统计基于 vercount，**无需注册**，部署后自动生效。如需更换为其他服务，可修改 `docs/.vitepress/theme/stats.ts`。

### 主题色

财经红金配色定义在 `docs/.vitepress/theme/style.css`，可按需调整 CSS 变量。
