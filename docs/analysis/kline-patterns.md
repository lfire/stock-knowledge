// 简单确定性"伪随机"：保证每次构建图形完全一致
function seeded(seed) {
  let s = seed
  return () => {
    s = (s * 9301 + 49297) % 233280
    return s / 233280
  }
}
// 由中枢收盘价数组生成 OHLC：开 ≈ 上一根收盘 ± 抖动；高低 = 包络
function toCandles(closes, opts = {}) {
  const { amp = 0.12, bias = 0, seed = 7 } = opts
  const rnd = seeded(seed)
  const out = []
  let prevClose = closes[0]
  for (let i = 0; i < closes.length; i++) {
    const c = closes[i]
    const open = i === 0 ? c - (rnd() - 0.5) * amp : prevClose + (rnd() - 0.5) * amp * 0.6
    const body = [Math.min(open, c), Math.max(open, c)]
    const high = body[1] + rnd() * amp + Math.max(0, bias)
    const low  = body[0] - rnd() * amp - Math.max(0, -bias)
    out.push([+open.toFixed(2), +c.toFixed(2), +low.toFixed(2), +high.toFixed(2)])
    prevClose = c
  }
  return out
}
// 由 K 线序列生成成交量：阳线 vs 阴线 + 节点强弱
function toVolumes(candles, profile = []) {
  return candles.map((k, i) => {
    const up = k[1] >= k[0]
    const base = up ? 1.0 : 0.85
    const factor = profile[i] ?? 1
    return Math.round(base * factor * 1.6e7)
  })
}

# K 线形态大全：四大类 103 种形态精解

> 本文系统整理了市面流传的「103 种炒股形态」，按 **强势、弱势、反转、震荡待涨** 四大类重新归纳，并为每一类的代表形态绘制 **交互式 K 线 / 分时图**（红涨绿跌，贴近同花顺等行情软件），帮助你理解形态背后的多空逻辑。
>
> ⚠️ 形态分析提供的是 **概率** 而非确定性，务必结合 [趋势位置](./technical-analysis)、[成交量与指标](./indicators) 和 [风险管理](../strategy/risk-management) 综合判断。本文仅供学习，不构成投资建议。

## 一、先读懂 K 线的「多空语言」

在看任何复杂形态之前，必须先读懂 **单根 K 线**。每根 K 线包含开盘、收盘、最高、最低四个价格：本站统一采用 A 股习惯——**收盘高于开盘为阳线（红），收盘低于开盘为阴线（绿）**；实体上下的细线为影线，代表盘中触及的最高 / 最低价。

<StockChart title="图例：阳线（红）与阴线（绿）" :candles="basic" :categories="['阳线 ▲', '阴线 ▼']" :height="260" />

**核心逻辑**：实体越长，多空一方力量越强；影线越长，说明该方向的进攻被有效反击。形态分析的本质，就是把若干根 K 线连起来，读出 **多空双方力量的此消彼长**。

| ---------------- | ------------------- | ------------------ | ------------------- |

| **弱势形态**     | 下跌趋势中          | 空方占优、趋势延续 | 观望 / 减仓离场     |
| **反转形态**     | 趋势末端（顶 / 底） | 多空力量发生逆转   | 见底布局 / 见顶撤退 |
| **震荡待涨形态** | 横盘 / 回调中       | 主力洗盘吸筹、蓄势 | 区间操作 / 突破跟进 |

> 同一个形态（如「长上影线」），出现在 **高位** 是见顶警讯，出现在 **回调末端** 却可能是洗盘蓄势——**位置决定性质**，这是看图的第一原则。

## 二、强势形态：顺势而为

强势形态出现在 **上升趋势** 中，核心特征是 **重心不断抬高、阳线多于阴线、回调不破支撑**。代表：红三兵、上升三法、一阳穿三线、出水芙蓉、涨停双响炮、W 底、上升通道等。

### 5 ｜ 红三兵

三根连续阳线，每根收盘价 **逐级抬高**，开盘价多落在前一根实体之内，是多方持续进攻的典型信号。

<StockChart title="红三兵：三连阳，重心逐级抬高（量价齐升）" :candles="redThree" :volumes="redThreeVol" :height="360" />

**逻辑**：连续三天买盘主动推高，且没有出现长上影线（抛压被持续消化），说明上攻意愿坚决。若出现在底部启动初期，意义更大。

### 9 ｜ 上升三法

一根大阳之后出现 **三根缩量小阴 / 小星线** 回调，但始终不跌破大阳的开盘价，随后再拉一根大阳创新高——典型的「上涨中继」。

<StockChart title="上升三法：放量大阳 → 缩量回调 → 再放量突破" :candles="risingThree" :volumes="risingThreeVol" :height="360" />

**逻辑**：小阴线缩量回调=获利盘有限、抛压轻；大阳重新放量突破=洗盘结束、趋势延续。

### 19 / 15 ｜ 一阳穿三线 · 出水芙蓉

一根长阳线 **自下而上一次性穿越多条均线**（如 5 / 10 / 20 日线）。当多条均线粘合后被一根放量大阳同时突破，即为「出水芙蓉」。

<StockChart title="一阳穿三线：长阳一次性贯穿 MA5/10/20（突破日放量）" :candles="pierceMA" :ma="pierceMaLines" :volumes="pierceVol" :height="380" />

**逻辑**：均线代表不同周期持仓者的平均成本，一根阳线同时收复所有成本线，意味着场内被套盘瞬间解套、压力转支撑，做多动能强劲。

### 6 ｜ 涨停双响炮

两根涨停（或接近涨停）的大阳线，中间夹着一根小幅整理的 K 线，形如「两声炮响」。

<StockChart title="涨停双响炮：两大阳夹一整理（双响放量）" :candles="doubleCannon" :volumes="doubleCannonVol" :height="360" />

**逻辑**：第一根涨停点火，回踩不破后第二根再封板，说明主力资金连续进攻、控盘力度强。

### 27 ｜ W 底（双重底）

价格两次探至相近低点后回升，颈线被有效突破，形如字母「W」。

<StockChart title="W 底（32 根日K）：前期下跌 → 底1(7.20) → 反弹至颈线 9.00 → 底2(7.30) → 放量突破" :candles="wBottomCandles" :volumes="wBottomVol" :categories="wBottomCats" :mark-lines="[{ y: 9.0, text: '颈线 9.00', color: '#ffb300' }, { y: 7.2, text: '前低支撑 7.20', color: '#3e9eff' }]" :zoom="true" :height="420" />

**走势拆解**（共 32 根日 K）：
1. **D1–D7 前期下跌**（10.80 → 7.80）：连续阴线放量，恐慌盘释放。
2. **D8 底 1 = 7.20**：长下影止跌，单日巨量。
3. **D9–D15 反弹至颈线**（7.55 → 8.95）：温和量能拉升，**在颈线 9.00 附近受阻**。
4. **D16–D22 颈线回落**（8.65 → 7.30）：缩量回踩，多方放慢节奏测试支撑。
5. **D23 底 2 = 7.30**：**略高于底 1**——空方力量衰竭的最关键证据。
6. **D24–D28 蓄势上攻**：阳线渐多、量能渐增。
7. **D29–D32 放量突破颈线**：大阳线放量突破 9.00，W 底成立。

**逻辑**：两次探底"低点不破"说明卖压已被消化；颈线被放量突破后，**理论涨幅 ≈ 颈线 − 底部 = 9.00 − 7.20 = 1.80**（目标价约 10.80，与图中 D32 的 ~10.20 接近）。买点：突破颈线确认日（D29）；止损：跌破底 2（7.30）。

### 12 ｜ 上升通道

股价在两条平行的上升趋势线之间运行：**下轨**提供支撑、**上轨**形成压力、**中轴**是价格回归的"重心"。重心稳步抬高即为上升通道。

<StockChart title="上升通道（40 根日K）：沿中轴震荡上行，回踩下轨买、触及上轨减" :candles="upChannelCandles" :volumes="upChannelVol" :categories="upChannelCats" :channel="upChannelBand" :zoom="true" :height="420" />

**走势拆解**（共 40 根日 K，中轴斜率 +0.10/日、通道宽度 1.40）：

| 节点 | 位置 | 行为 | 操作含义 |
| --- | --- | --- | --- |
| **D7** | 贴近下轨 | 回踩支撑、放量企稳（长下影线） | **第一次买点** |
| **D13/D15** | 触及上轨 | 缩量滞涨、收阴 | **第一次减仓点** |
| **D21** | 再次贴近下轨 | 二次回踩、再度企稳 | **第二次买点** |
| **D29/D31** | 再次触及上轨 | 触压力回落 | **第二次减仓点** |

**逻辑**：通道是"上升趋势 + 周期性波动"的几何表达。操作要点：
1. **回踩下轨买入**（支撑反复有效）；
2. **触及上轨减仓**（压力反复存在）；
3. **跌破下轨**是趋势破坏的第一信号，应离场观望；
4. **放量突破上轨** = 加速段，可顺势加仓而非减仓。

## 三、弱势形态：及时避险

弱势形态出现在 **下跌趋势** 中，核心特征是 **重心不断下移、阴线多于阳线、反弹无力**。代表：三只乌鸦、黑三兵、下降通道、射击之星、断头铡刀等。

### 34 / 35 ｜ 三只乌鸦 · 黑三兵

三根连续阴线，收盘价 **逐级走低**，是红三兵的镜像。出现在高位是强烈的派发出货信号。

<StockChart title="三只乌鸦：三连阴，重心逐级走低（放量下跌）" :candles="threeCrows" :volumes="threeCrowsVol" :height="360" />

**逻辑**：连续三天卖盘主动打压且无明显下影线，说明承接乏力、空方完全主导。「黑三兵」与之类似，强调阴线绵延、节节败退。

### 37 ｜ 射击之星

在上涨高位出现一根 **带长上影线的小实体** K 线：盘中冲高，但尾盘被大幅打回。

<StockChart title="射击之星：高位长上影 = 见顶警讯" :candles="shootingStar" :height="280" />

**逻辑**：上影线越长，说明高位抛压越重、追高资金被套。它与底部的「仙人指路」长相相似，区别全在 **位置**。

### 43 ｜ 断头铡刀

一根长阴线 **自上而下一次性跌破多条均线**，是「一阳穿三线」的反向形态。

<StockChart title="断头铡刀：长阴一次性跌破多条均线（破位日巨量）" :candles="guillotine" :ma="guillotineMaLines" :volumes="guillotineVol" :height="380" />

**逻辑**：所有周期的持仓成本一日内全部被击穿，技术形态瞬间转坏，止损盘踩踏，后市往往加速下行。

### 36 ｜ 下降通道

股价在两条平行的下降线之间运行，反弹不过上轨，逢高即是减仓机会。

<StockChart title="下降通道（40 根日K）：沿中轴震荡下行，反弹触上轨即减仓" :candles="downChannelCandles" :volumes="downChannelVol" :categories="downChannelCats" :channel="downChannelBand" :zoom="true" :height="420" />

**逻辑**：只要不放量突破上轨，下跌趋势就未结束，切忌在通道内「抄底」摊低成本。

## 四、反转形态：捕捉拐点

反转形态出现在 **趋势末端**，标志多空力量的逆转。底部反转是布局机会，顶部反转是离场信号。代表：早晨十字星、刺透线、乌云盖顶、穿头破脚、V 型反转、圆弧顶 / 底等。

### 52 ｜ 早晨十字星（底部反转）

下跌末端出现「大阴线 → 跳空十字星 → 大阳线」三段式结构，是经典的见底信号。

<StockChart title="早晨十字星：恐慌放量 → 十字星缩量 → 反攻放量" :candles="morningStar" :volumes="morningStarVol" :height="360" />

**逻辑**：第二根十字星表示多空陷入平衡、恐慌盘衰竭；第三根大阳线确认多方反攻，黎明到来。

### 54 / 69 ｜ 刺透线 vs 乌云盖顶（一对镜像）

二者都是两根 K 线组合，方向相反。

**刺透线（底部看涨）**——大阴之后，次日低开高走，阳线收复前阴实体一半以上。

<StockChart title="刺透线：阳线深入前阴实体 1/2 以上（底部）" :candles="piercing" :height="260" />

**乌云盖顶（顶部看跌）**——大阳之后，次日高开低走，阴线深入前阳实体一半以上。

<StockChart title="乌云盖顶：阴线深入前阳实体 1/2 以上（顶部）" :candles="darkCloud" :height="260" />

**逻辑**：刺入对方实体越深，反转力度越强；插入超过实体全部即演变为更强的「穿头破脚（吞没）」。

### 61 ｜ 穿头破脚（看涨吞没）

一根大阳线 **完全包住** 前一根小阴线（最高、最低均超越），底部出现是强烈的反转信号。

<StockChart title="穿头破脚：大阳完全吞没前一根小阴（吞没日放量）" :candles="engulfing" :volumes="engulfingVol" :height="360" />

**逻辑**：多方一天的攻击就抹平了前一日空方的全部成果，力量对比骤然反转。

### 68 ｜ V 型反转

价格急跌后不作整理、快速拉起，走出尖底「V」字，多由突发利好或超跌反弹驱动。

<StockChart title="V 型反转（24 根日K）：连续阴线急跌 → 单日恐慌底 → 连续阳线急涨" :candles="vReversalCandles" :volumes="vReversalVol" :categories="vReversalCats" :mark-lines="[{ y: 5.8, text: '尖底 5.80', color: '#3e9eff' }]" :zoom="true" :height="420" />

**走势拆解**（共 24 根日 K）：
1. **D1–D12 急跌段**：12 连阴，重心从 12.80 急速跌至 6.20，**量能逐日放大**（恐慌出货）。
2. **D13 尖底 = 5.80**：单日长下影十字星，最低 5.80，巨量止跌（量能见峰）。
3. **D14–D24 急涨段**：11 连阳几乎不作整理，重心从 6.40 快速拉回 12.85，**量能呈衰减式放量**。

**逻辑**：V 形是恐慌情绪一日见底、信心瞬间逆转的极端形态。**反转迅猛、不给低吸机会**，多由突发利好或超跌反弹驱动；顶部的倒 V（尖顶）则极其凶险，常见于业绩暴雷。

### 83 ｜ 圆弧底

价格以平滑弧线缓慢转向，温和筑底后弧形回升，趋势持续性较好，适合中线布局。

<StockChart title="圆弧底（40 根日K）：弧形缓跌 → 底部地量 → 弧形缓涨" :candles="roundBottomCandles" :volumes="roundBottomVol" :categories="roundBottomCats" :zoom="true" :height="420" />

**走势拆解**（共 40 根日 K，价格围绕一条余弦弧线）：
- **D1–D13 弧形缓跌**：阴线虽多但实体逐日缩小，**量能逐步萎缩**（卖盘衰竭）。
- **D14–D27 底部地量**：价格在 7.0 附近反复磨底，**成交量地量**——这是圆弧底最关键的特征。
- **D28–D40 弧形缓涨**：阳线渐多、实体渐大，**量能温和放大**，重心稳步回升至 11.0。

**逻辑**：弧形转向说明多空力量是**渐变而非突变**——主力悄悄吸筹、不留痕迹。一旦弧形确立，**趋势持续性强**，适合中长线布局（顶部的圆弧顶则是反过来的派发）。

## 五、震荡待涨形态：洗盘与蓄势

震荡待涨形态出现在 **横盘或回调** 阶段，本质多是 **主力洗盘吸筹、蓄势待发**。核心特征是 **下有支撑、缩量整理、重心不破**。代表：箱体震荡、圆弧底、黄金坑、三重底、仙人指路、空中加油等。

### 85 ｜ 箱体震荡

股价在水平的箱顶（压力）与箱底（支撑）之间反复运行，方向待选；放量突破箱顶则变盘向上。

<StockChart title="箱体震荡（36 根日K）：箱顶/箱底之间往返 3 次后放量突破" :candles="boxCandles" :volumes="boxVol" :categories="boxCats" :mark-lines="[{ y: 10.0, text: '箱顶（压力）10.00', color: '#ef232a' }, { y: 8.0, text: '箱底（支撑）8.00', color: '#16b15a' }]" :zoom="true" :height="420" />

**走势拆解**（共 36 根日 K）：
- **D1–D13 第一轮**：箱底 8.10 上行至箱顶 9.95，触顶回落到 8.15。
- **D14–D25 第二轮**：再次箱底→箱顶→箱底，每次接近箱顶都收阴回落。
- **D26–D31 第三轮**：箱底 8.20 上攻至箱顶 9.95。
- **D32–D36 放量突破**：连续 5 根中阳，价格突破 10.00 箱顶，重心抬升至 11.40，**突破日量能放大 3 倍**。

**逻辑**：箱体是多空僵持的"蓄能区"——震荡时间越长、突破后的爆发力越强。关键看 **突破方向 + 量能确认**：放量向上破箱顶则待涨成立，跌破箱底则形态转弱。箱内操作以"贴箱底买、贴箱顶卖"高抛低吸为主。

### 86 ｜ 黄金坑

上涨途中突然急跌「挖坑」（制造恐慌、洗出浮筹），随后又快速强势收复，形如一个坑。

<StockChart title="黄金坑（28 根日K）：上涨 → 急跌挖坑 → 坑底巨量 → 强势收复 → 突破前高" :candles="goldenPitCandles" :volumes="goldenPitVol" :categories="goldenPitCats" :mark-lines="[{ y: 10.0, text: '前高 / 颈线 10.00', color: '#ffb300' }, { y: 7.0, text: '坑底 7.00', color: '#3e9eff' }]" :zoom="true" :height="420" />

**走势拆解**（共 28 根日 K）：
1. **D1–D7 前期上涨**（8.00 → 10.00）：温和放量上行，建立**前高 10.00**。
2. **D8–D13 急跌挖坑**（10.00 → 7.20）：连续 6 根阴线（含 1 根长阴），**量能逐日放大**——主力主动制造恐慌。
3. **D14 坑底 7.00**：长下影线 + **当日巨量**（量能见峰），洗盘到位。
4. **D15–D21 强势收复**（7.55 → 10.25）：连续阳线，**两根放量长阳**直接收复前高 10.00——这是黄金坑成立的关键确认。
5. **D22–D28 突破前高继续上行**（10.50 → 11.60）：洗盘后启动主升段。

**逻辑**：坑越深、收复越快、收复时越放量，洗盘越彻底，后续上攻空间越大。**判断要点**：能否 **快速收复失地并突破前高**——若反弹无力或迟迟无法回到坑前水平，则不是黄金坑而是真破位。

### 87 ｜ 三重底

价格三次探至相近低点均获支撑，形成三个底，突破颈线后看涨。

| 前期下跌 → 底 1 | D1–D7 | 9.50 → **7.00** | 下跌放量、底部巨量 |

### 84 ｜ 仙人指路

回调末端出现一根 **带长上影线的小阳 / 小实体**：盘中向上试盘触及压力后回落，但整体重心未破。它与高位的「射击之星」形态几乎一样，但出现在 **回调 / 起涨位置**，是主力试盘、确认抛压后再度发动的信号——再次印证「位置决定性质」。

> 提示：仙人指路的单根 K 线形态可直接参考上文「射击之星」的图形，差别只在 **它出现在回调低位而非上涨高位**。

## 六、实战使用的四条原则

形态再多，落到实战只需把握四点：

1. **先定位置，再认形态**：同一形态在顶部和底部含义相反。永远先判断当前处于趋势的哪个阶段。
2. **形态必须配合量能**：突破、反转要 **放量**，洗盘、蓄势要 **缩量**。无量的形态可信度大打折扣。
3. **形态需要后续确认**：单根 / 单组 K 线只是「信号」，要等次日 / 突破颈线等动作 **确认** 后再行动，避免诱多诱空。
4. **设好止损，控制仓位**：形态提供的是概率优势而非必然，任何一笔交易都应预设止损位，参考 [风险管理](../strategy/risk-management)。

> **一句话总结**：K 线形态是「多空博弈」留下的脚印。看懂脚印的关键，不是死记 103 个名字，而是理解每个形态背后 **多方与空方谁在进攻、谁在防守、谁更有力量**。
