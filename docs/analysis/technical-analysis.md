# 技术分析入门

## 什么是技术分析

**技术分析（Technical Analysis）** 是通过研究历史价格和成交量数据，来预测未来价格走势的方法。它建立在三大假设之上：

1. **市场行为包容一切**：所有信息都已反映在价格中。
2. **价格沿趋势运动**：趋势一旦形成，更倾向于延续。
3. **历史会重演**：价格形态会反复出现，因为人性不变。

## K 线图基础

K 线（蜡烛图）是技术分析的核心工具，每根 K 线包含四个价格：

- **开盘价**、**收盘价**、**最高价**、**最低价**

阳线（收盘 > 开盘）通常用红色（A 股）或绿色（欧美）表示，阴线则相反。

```
   ┃  ← 上影线（最高价）
  ┏━┓
  ┃ ┃ ← 实体（开盘价与收盘价之间）
  ┗━┛
   ┃  ← 下影线（最低价）
```

## 支撑与阻力

- **支撑位**：股价下跌时容易获得买盘支撑而止跌的价格区域。
- **阻力位**：股价上涨时容易遇到卖压而回落的价格区域。

支撑与阻力可以相互转化：阻力一旦被有效突破，往往转变为新的支撑。

## 趋势的三种状态

| 趋势     | 特征                       | 操作倾向           |
| -------- | -------------------------- | ------------------ |
| 上升趋势 | 高点不断抬高，低点不断抬高 | 逢低买入、持有     |
| 下降趋势 | 高点不断降低，低点不断降低 | 观望、减仓         |
| 横盘震荡 | 在区间内反复波动           | 高抛低吸或等待突破 |

## 注意事项

技术分析并非"水晶球"，它提供的是**概率**而非确定性。务必结合 [基本面分析](./fundamental-analysis) 与 [风险管理](../strategy/risk-management) 综合判断。
