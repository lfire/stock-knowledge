# 基本面分析

## 什么是基本面分析

**基本面分析（Fundamental Analysis）** 通过研究公司的财务状况、经营能力、行业前景与宏观环境，来评估股票的**内在价值**，进而判断当前股价是被高估还是低估。

核心理念：**价格围绕价值波动**。当价格显著低于价值时，便存在投资机会。

## 自上而下的分析框架

1. **宏观经济**：经济周期、利率、通胀、政策导向。
2. **行业分析**：行业空间、竞争格局、景气度。
3. **公司分析**：商业模式、护城河、管理层、财务质量。

## 三大财务报表

| 报表 | 反映内容 | 关注重点 |
| --- | --- | --- |
| 利润表 | 一段时间的经营成果 | 营收增长、净利润、毛利率 |
| 资产负债表 | 某时点的资产与负债 | 资产质量、负债率、现金 |
| 现金流量表 | 现金的流入与流出 | 经营性现金流是否健康 |

## 关键财务指标

- **ROE（净资产收益率）**：净利润 ÷ 净资产，衡量股东资金的盈利效率，巴菲特最看重的指标之一。
- **毛利率 / 净利率**：反映产品竞争力与盈利能力。
- **资产负债率**：衡量财务杠杆与偿债风险。
- **营收/利润增长率**：反映成长性。

## 估值方法

- **相对估值**：PE、PB、PS 等，与同行或历史水平对比。
- **绝对估值**：DCF（现金流折现）等，测算内在价值。

## 小结

基本面分析回答"买什么"的问题，技术分析回答"何时买"的问题，两者结合效果更佳。
