# 如何快速读懂一份财报

财报动辄上百页，但对普通投资者来说，抓住关键信息即可。本文给出一条高效的阅读路径。

## 先看三张报表的"主角"

| 报表       | 一句话看懂         | 重点科目                   |
| ---------- | ------------------ | -------------------------- |
| 利润表     | 公司赚不赚钱       | 营业收入、净利润、毛利率   |
| 资产负债表 | 公司家底如何       | 货币资金、有息负债、净资产 |
| 现金流量表 | 利润是否"真金白银" | 经营活动现金流净额         |

## 再算四个关键指标

1. **营收与净利润增速**：判断成长性，最好连续多年稳定增长。
2. **毛利率**：反映产品竞争力，越高且越稳定越好。
3. **ROE（净资产收益率）**：长期 > 15% 通常是优秀企业的标志。
4. **经营性现金流 / 净利润**：接近或大于 1，说明利润含金量高。

## 警惕这些"危险信号"

- 净利润增长但经营性现金流持续为负 → 利润可能是"纸面富贵"。
- 应收账款增速远超营收 → 可能存在压货或回款风险。
- 有息负债高企且现金紧张 → 偿债压力大。

## 实操建议

::: tip 阅读顺序
管理层讨论与分析（MD&A）→ 三张主表 → 关键指标计算 → 重要附注（关注关联交易、商誉、存货）。
:::

读财报的目的不是成为会计师，而是判断"这门生意好不好、这家公司稳不稳"。多读几家同行业公司的财报，对比之下会更有体感。
