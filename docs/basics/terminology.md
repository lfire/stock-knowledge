# 常见交易术语

初学者常被各种术语困扰，下面整理了最常用的概念，便于随时查阅。

## 价格相关

- **开盘价 / 收盘价**：当日第一笔与最后一笔成交价。
- **最高价 / 最低价**：当日成交的最高与最低价格。
- **涨停 / 跌停**：股价达到当日允许波动的上限或下限。

## 估值相关

- **市盈率（PE）**：股价 ÷ 每股收益，衡量股价相对盈利的高低。
- **市净率（PB）**：股价 ÷ 每股净资产，常用于评估资产型公司。
- **市值**：股价 × 总股本，代表公司在市场中的总价值。

## 交易相关

- **成交量 / 成交额**：一定时间内的交易股数 / 金额，反映市场活跃度。
- **换手率**：成交量 ÷ 流通股本，衡量股票交易的活跃程度。
- **多头 / 空头**：看涨的买方 / 看跌的卖方。
- **做多 / 做空**：预期上涨而买入 / 预期下跌而卖出（融券）。

## 账户相关

- **建仓 / 加仓 / 减仓 / 平仓**：建立、增加、减少、了结持仓的操作。
- **满仓 / 空仓 / 半仓**：资金全部买入 / 全部为现金 / 部分买入。
- **套牢**：买入后股价下跌，被迫持有的状态。

## 指数相关

- **大盘**：通常指代表性指数（如上证指数），反映市场整体走势。
- **蓝筹股**：业绩稳定、市值大的优质龙头公司股票。
- **题材股 / 概念股**：因某种热点题材而被关注的股票。
