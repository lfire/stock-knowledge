# 什么是股票

## 定义

**股票（Stock）** 是股份公司发行的所有权凭证。当你购买一家公司的股票，就意味着你成为了这家公司的股东，拥有公司的一部分所有权。

## 股票代表什么

持有股票通常意味着你享有以下权利：

- **分红权**：公司盈利后，可按持股比例分配股息（红利）。
- **投票权**：在股东大会上对公司重大事项进行投票。
- **资本增值**：当股价上涨时，卖出股票可获得差价收益。

## 股票的分类

| 分类维度 | 类型 | 说明 |
| --- | --- | --- |
| 按权利 | 普通股 | 享有投票权和分红权，是最常见的股票 |
| 按权利 | 优先股 | 优先获得固定股息，通常无投票权 |
| 按市值 | 大盘股 / 中小盘股 | 按公司总市值规模划分 |
| 按行业 | 周期股 / 成长股 / 价值股 | 按公司商业模式与成长性划分 |

## 股票与其他投资品的区别

- **股票 vs 债券**：股票是所有权，债券是债权（借钱给公司，到期还本付息）。
- **股票 vs 基金**：基金是把钱交给基金经理打包投资多只股票/债券，分散风险。

## 小结

股票本质上是企业所有权的一部分。理解"买股票就是买公司"这一核心理念，是建立正确投资观的第一步。
