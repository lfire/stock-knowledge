# 价值投资

## 核心理念

**价值投资（Value Investing）** 由本杰明·格雷厄姆提出，并由沃伦·巴菲特发扬光大。其核心是：以**低于内在价值的价格**买入优质公司，并长期持有，分享企业成长的收益。

> "价格是你支付的，价值是你得到的。" —— 沃伦·巴菲特

## 三大基石

1. **安全边际**：买入价格要明显低于内在价值，为判断失误预留缓冲。
2. **能力圈**：只投资自己能理解的行业与公司。
3. **市场先生**：把市场波动看作情绪化的报价者，利用而非被其左右。

## 寻找优质公司的标准

- **宽阔的护城河**：品牌、网络效应、成本优势、转换成本等竞争壁垒。
- **持续的盈利能力**：长期稳定的高 ROE。
- **健康的财务**：低负债、充裕的现金流。
- **优秀且诚信的管理层**。

## 价值投资 vs 投机

| 维度 | 价值投资 | 投机 |
| --- | --- | --- |
| 关注点 | 企业内在价值 | 价格短期波动 |
| 持有周期 | 中长期 | 短期 |
| 决策依据 | 基本面 | 情绪、消息 |
| 风险来源 | 看错企业 | 看错价格、追涨杀跌 |

## 实践要点

- 坚持长期视角，避免频繁交易。
- 在市场恐慌、优质资产被错杀时勇于买入。
- 定期复盘持仓逻辑，逻辑破坏时果断卖出。
